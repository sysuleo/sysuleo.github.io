[problems solution] windows cmd cd无效问题

Posted on 2021-04-22 In problems solution Views:

问题描述

在windows下， 打开cmd.exe, 默认进入到当前用户主目录下， 发现除了C:\盘， 可以通过 cd、dir 指令 去到C:\盘的各个目录外， 不能通过 cd e: 进入到其他盘

solution

  1. 在 cd 和盘符之间加上 /d
  2. 直接用 盘符 e:
