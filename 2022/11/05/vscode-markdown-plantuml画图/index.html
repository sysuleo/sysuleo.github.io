需要#

  1. Java环境（PlantUML需要java环境）
  2. VSCode （不必多说）
  3. VSCode插件
  4. Graphviz软件(PlantUML的图形预览)

安装#

  1. 安装java
  2. VSCode商店安装如下插件:
    • Markdown Preview Enhanced
    • Markdown All In One
    • PlantUml
  3. 安装Graphviz软件
    • 下载Graphviz windows版
    • 安装时注意选中 Add Graphviz to the system PATH for all users，若未选中自己配置下环境变量
