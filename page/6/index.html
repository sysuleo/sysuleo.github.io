Python жҸҗдҫӣзҡ„и§ЈеҶіж–№жЎҲжҳҜ Unicode еӯ—з¬ҰдёІ ( str )еҜ№иұЎпјҢ Unicode еҸҜд»ҘиЎЁзӨәеҗ„з§Қеӯ—з¬ҰпјҢж— йңҖе…іеҝғзј–з ҒгҖӮз„¶иҖҢеӯҳеӮЁжҲ–иҖ…зҪ‘з»ңйҖҡи®Ҝж—¶пјҢеӯ—з¬ҰдёІеҜ№иұЎдёҚеҸҜйҒҝе…ҚиҰҒ еәҸеҲ—еҢ– жҲҗеӯ—иҠӮеәҸеҲ—гҖӮдёәжӯӨпјҢ Python йўқеӨ–жҸҗдҫӣдәҶ еӯ—иҠӮеәҸеҲ—еҜ№иұЎвҖ”вҖ” bytes гҖӮ

еҰӮдёҠеӣҫпјҢ str еҜ№иұЎз»ҹдёҖиЎЁзӨәдёҖдёӘ еӯ—з¬ҰдёІ пјҢдёҚйңҖиҰҒе…іеҝғзј–з Ғпјӣи®Ўз®—жңәйҖҡиҝҮ еӯ—иҠӮеәҸеҲ— дёҺеӯҳеӮЁд»ӢиҙЁе’ҢзҪ‘з»ңд»ӢиҙЁжү“дәӨйҒ“пјҢеӯ—иҠӮеәҸеҲ—з”ұ bytes еҜ№иұЎиЎЁзӨәпјӣеӯҳеӮЁжҲ–дј иҫ“ str еҜ№иұЎж—¶пјҢйңҖиҰҒе°Ҷе…¶ еәҸеҲ—еҢ– жҲҗеӯ—иҠӮеәҸеҲ—пјҢеәҸеҲ—еҢ–иҝҮзЁӢд№ҹжҳҜ зј–з Ғ зҡ„иҝҮзЁӢгҖӮ

еҜ№иұЎз»“жһ„#

bytes еҜ№иұЎз”ЁдәҺиЎЁзӨәз”ұиӢҘе№Іеӯ—иҠӮз»„жҲҗзҡ„ еӯ—иҠӮеәҸеҲ— д»ҘеҸҠзӣёе…ізҡ„ ж“ҚдҪң пјҢе№¶дёҚе…іеҝғеӯ—иҠӮеәҸеҲ—зҡ„ еҗ«д№ү гҖӮеӣ жӯӨпјҢ bytes еә”иҜҘдёҖз§Қ еҸҳй•ҝ гҖҒ дёҚеҸҜеҸҳ еҜ№иұЎ пјҢеҶ…йғЁз”ұ C ж•°з»„ е®һзҺ°гҖӮеҰӮдёӢеӣҫ:

  1. ob_sval еӯ—иҠӮеәҸеҲ—еҜ№иұЎ PyBytesObject дёӯпјҢзЎ®е®һи—ҸзқҖдёҖдёӘеӯ—з¬Ұж•°з»„ ob_sval гҖӮжіЁж„ҸеҲ° ob_sval ж•°з»„й•ҝеәҰе®ҡд№үдёә 1 пјҢиҝҷжҳҜ C иҜӯиЁҖдёӯе®ҡд№ү еҸҳй•ҝж•°з»„ зҡ„жҠҖе·§пјҲob_svalеӯҳеӮЁзҡ„жҳҜең°еқҖпјүгҖӮ

  2. ob_snash ob_shash пјҢе®ғз”ЁдәҺдҝқеӯҳеӯ—иҠӮеәҸеҲ—зҡ„ е“ҲеёҢеҖј гҖӮ з”ұдәҺи®Ўз®— bytes еҜ№иұЎе“ҲеёҢеҖјйңҖиҰҒйҒҚеҺҶе…¶еҶ…йғЁзҡ„еӯ—з¬Ұж•°з»„пјҢејҖй”ҖзӣёеҜ№иҫғеӨ§гҖӮеӣ жӯӨпјҢ Python з¬¬дёҖж¬Ўи®Ўз®— е“ҲеёҢеҖјж—¶пјҢйҖүжӢ© е°Ҷе“ҲеёҢеҖјзј“еӯҳеҲ° ob_shashеӯ—ж®өдёӯпјҢд»Ҙ з©әй—ҙжҚўж—¶й—ҙпјҢйҒҝе…ҚйҮҚеӨҚи®Ўз®—гҖӮ

  3. ob_size жҜҸдёӘPyVarObjectеҶ…йғЁйғҪжңүдёӘ ob_sizeеӯ—ж®өпјҢPyBytesObjectдҪҝз”ЁжӯӨеӯ—ж®өеӯҳеӮЁеӨ§е°ҸдҝЎжҒҜд»Ҙ дҝқжҢҒlenпјҲпјүж“ҚдҪңзҡ„OпјҲ1пјүж—¶й—ҙеӨҚжқӮеәҰпјҢе№¶и·ҹиёӘйқһasciiеӯ—з¬ҰдёІзҡ„еӨ§е°ҸпјҲеҶ…йғЁеҸҜд»Ҙдёәз©әеӯ—з¬Ұпјү

з©әеҜ№иұЎж ·дҫӢ#

Python дёәеҫ…еӯҳеӮЁзҡ„еӯ—иҠӮеәҸеҲ— йўқеӨ–еҲҶй…ҚдёҖдёӘеӯ—иҠӮпјҢз”ЁдәҺеңЁжң«е°ҫеӨ„дҝқеӯҳ \0 пјҢд»Ҙдҫҝе…је®№ C еӯ—з¬ҰдёІгҖӮд»ҺдёҠеӣҫеҸҜд»ҘзңӢеҮәпјҢе°ұз®—з©ә bytes еҜ№иұЎ( b'' )д№ҹжҳҜиҰҒеҚ з”ЁеҶ…еӯҳз©әй—ҙзҡ„пјҢиҮіе°‘еҸҳй•ҝеҜ№иұЎ е…¬е…ұеӨҙйғЁ жҳҜе°‘дёҚдәҶзҡ„гҖӮ

bytes еҜ№иұЎеҚ з”Ёзҡ„еҶ…еӯҳз©әй—ҙеҸҜеҲҶдёәд»ҘдёӢдёӘйғЁеҲҶиҝӣиЎҢи®Ўз®—пјҡ

  • PyVarObjectе…¬е…ұеӨҙйғЁ 24 еӯ—иҠӮпјҢob_refcnt гҖҒ ob_type гҖҒ ob_size жҜҸдёӘеӯ—ж®өеҗ„еҚ з”Ё 8 еӯ—иҠӮпјӣ
  • е“ҲеёҢеҖј ob_shash еҚ з”Ё 8 еӯ—иҠӮпјӣ
  • еӯ—иҠӮеәҸеҲ—жң¬иә«пјҢеҒҮи®ҫжҳҜ n еӯ—иҠӮпјӣ
  • йўқеӨ– 1 еӯ—иҠӮз”ЁдәҺеӯҳеӮЁжң«е°ҫеӨ„зҡ„ \0 пјӣ

еӣ жӯӨпјҢbytes еҜ№иұЎз©әй—ҙи®Ўз®—е…¬ејҸдёә 24+8+n+124+8+n+1пјҢеҚі 33+n33+nпјҢе…¶дёӯ n дёәеӯ—иҠӮеәҸеҲ—й•ҝеәҰ(д№ҹжҳҜlenзҡ„еҸ–еҖј)гҖӮ з»ҸиҝҮдёҠйқўзҡ„еӯҰд№ пјҢжҲ‘д»¬еҸҜд»ҘзҹҘйҒ“ len(byteеҜ№иұЎ) = nпјҢlenжҳҫзӨәзҡ„еҸӘжҳҜ ob_sizeеӯ—ж®өзҡ„еҖјпјҢиҖҢbytesеҜ№иұЎзңҹе®һеҚ з”ЁеҶ…еӯҳйҮҸиҝҳ йңҖиҰҒеҠ  33.

еҜ№иұЎиЎҢдёә#

еҜ№иұЎзҡ„иЎҢдёәз”ұеҜ№иұЎзҡ„ зұ»еһӢ еҶіе®ҡпјҢеӣ иҖҢжҲ‘д»¬йңҖиҰҒеҲ° bytes зұ»еһӢеҜ№иұЎ(PyBytes_Type)дёӯеҜ»жүҫзӯ”жЎҲгҖӮеңЁ Objects/bytesobject.c жәҗз Ғж–Үд»¶дёӯпјҢжҲ‘д»¬жүҫеҲ° bytes зұ»еһӢеҜ№иұЎ зҡ„е®ҡд№үпјҡ

bytes еҜ№иұЎеұ…з„¶ж”ҜжҢҒж•°жҚ®ж“ҚдҪңпјҹbytes_as_number з»“жһ„дҪ“дёӯеҸӘе®ҡд№үдәҶдёҖдёӘж“ҚдҪңвҖ”вҖ” жЁЎиҝҗз®— ( % )пјҡ з”ұжӯӨеҸҜи§ҒпјҢ bytes еҜ№иұЎеҸӘжҳҜ еҖҹз”Ё % иҝҗз®—з¬Ұе®һзҺ°еӯ—з¬ҰдёІж јејҸеҢ–пјҢи°ҲдёҚдёҠж”ҜжҢҒж•°еҖјиҝҗз®—пјҢиҷҡжғҠдёҖеңәпјҡ

еәҸеҲ—еһӢж“ҚдҪң#

дј—жүҖе‘ЁзҹҘпјҢ bytes жҳҜ еәҸеҲ—еһӢеҜ№иұЎ пјҢеәҸеҲ—еһӢж“ҚдҪңжүҚжҳҜз ”з©¶йҮҚзӮ№гҖӮжҲ‘д»¬еңЁ bytes_as_sequence з»“жһ„дҪ“дёӯжүҫеҲ°зӣёе…іе®ҡд№үпјҡ

з”ұжӯӨеҸҜи§ҒпјҢ bytes ж”ҜжҢҒзҡ„ еәҸеҲ—еһӢж“ҚдҪң еҢ…жӢ¬д»ҘдёӢ 5 дёӘпјҡ - sq_length пјҢжҹҘиҜўеәҸеҲ—й•ҝеәҰпјӣ - sq_concat пјҢе°ҶдёӨдёӘеәҸеҲ—еҗҲе№¶дёәдёҖдёӘпјӣ - sq_repeat пјҢе°ҶеәҸеҲ—йҮҚеӨҚеӨҡж¬Ўпјӣ - sq_item пјҢеҸ–еҮәз»ҷе®ҡдёӢж ҮеәҸеҲ—е…ғзҙ пјӣ - sq_containsпјҢеҢ…еҗ«е…ізі»еҲӨж–ӯпјӣ

й•ҝеәҰ#

жңҖз®ҖеҚ•зҡ„еәҸеҲ—еһӢж“ҚдҪңжҳҜ й•ҝеәҰжҹҘиҜў пјҢзӣҙжҺҘиҝ”еӣһ ob_size еӯ—ж®өеҚіеҸҜпјҡ

еҗҲе№¶#

еҗҲе№¶ж“ҚдҪңе°ҶдёӨдёӘ bytes еҜ№иұЎжӢјжҺҘжҲҗдёҖдёӘпјҢз”ұ bytes_concat еҮҪж•°еӨ„зҗҶпјҡ

bytes_concat еҮҪж•°йҖ»иҫ‘еҫҲзӣҙзҷҪпјҢе°ҶдёӨдёӘ bytes еҜ№иұЎзҡ„зј“еҶІеҢәжӢ·иҙқеҲ°дёҖиө·еҪўжҲҗж–° bytes еҜ№иұЎгҖӮ

ж•°жҚ®жӢ·иҙқзҡ„йҷ·йҳұ#

иҖғеҜҹд»ҘдёӢиЎЁиҫҫејҸвҖ”вҖ”еҗҲе№¶ 3 дёӘ bytes еҜ№иұЎпјҡ

иҝҷдёӘиҜӯеҸҘжү§иЎҢж—¶пјҢеҲҶжҲҗдёӨжӯҘиҝӣиЎҢеҗҲе№¶пјҡе…Ҳе°Ҷ a е’Ң b еҗҲе№¶пјҢеҫ—еҲ°дёҙж—¶з»“жһң t пјҢеҶҚе°Ҷ t е’Ң c еҗҲе№¶еҫ—еҲ°жңҖз»Ҳз»“жһң result пјҡ иҝҷдёӘиҝҮзЁӢдёӯпјҢa е’Ң b зҡ„ж•°жҚ®йңҖиҰҒиў«жӢ·иҙқдёӨйҒҚпјҒ

еҗҲе№¶ n дёӘ bytes еҜ№иұЎпјҢеӨҙдёӨдёӘеҜ№иұЎйңҖиҰҒжӢ·иҙқ n-1 ж¬ЎпјҢеҸӘжңүжңҖеҗҺдёҖдёӘеҜ№иұЎдёҚйңҖиҰҒйҮҚеӨҚжӢ·иҙқгҖӮе№іеқҮдёӢжқҘпјҢжҜҸдёӘеҜ№иұЎеӨ§зәҰиҰҒжӢ·иҙқ n/2 ж¬ЎпјҒ

еҶ…е»әж–№жі• join#

bytes еҜ№иұЎжҸҗдҫӣдәҶдёҖдёӘеҶ…е»әж–№жі• join пјҢеҸҜй«ҳж•ҲеҗҲе№¶еӨҡдёӘ bytes еҜ№иұЎпјҡ join ж–№жі•еҜ№ж•°жҚ®жӢ·иҙқиҝӣиЎҢдәҶдјҳеҢ–пјҡе…ҲйҒҚеҺҶеҫ…еҗҲе№¶еҜ№иұЎпјҢи®Ўз®—жҖ»й•ҝеәҰпјӣз„¶еҗҺж №жҚ®жҖ»й•ҝеәҰ еҲӣе»әзӣ®ж ҮеҜ№иұЎпјӣжңҖеҗҺеҶҚ йҒҚеҺҶеҫ…еҗҲе№¶еҜ№иұЎпјҢйҖҗдёҖжӢ·иҙқж•°жҚ®гҖӮиҝҷж ·дёҖжқҘпјҢжҜҸдёӘеҜ№иұЎеқҮеҸӘйңҖжӢ·иҙқдёҖж¬ЎпјҢи§ЈеҶідәҶйҮҚеӨҚжӢ·иҙқзҡ„йҷ·йҳұгҖӮ

еӯ—з¬Ұзј“еҶІжұ #

дёәдәҶдјҳеҢ–еҚ•еӯ—иҠӮ bytes еҜ№иұЎ(д№ҹеҸҜз§°дёә еӯ—з¬ҰеҜ№иұЎ )зҡ„еҲӣе»әж•ҲзҺҮпјҢ Python еҶ…йғЁз»ҙжҠӨдәҶдёҖдёӘ еӯ—з¬Ұзј“еҶІжұ  пјҡ

Python еҶ…йғЁ еҲӣе»әеҚ•еӯ—иҠӮ bytes еҜ№иұЎж—¶пјҢе…ҲжЈҖжҹҘзӣ®ж ҮеҜ№иұЎжҳҜеҗҰе·ІеңЁзј“еҶІжұ дёӯгҖӮPyBytes_FromStringAndSize еҮҪж•°жҳҜиҙҹиҙЈеҲӣе»ә bytes еҜ№иұЎзҡ„йҖҡз”ЁжҺҘеҸЈпјҢеҗҢж ·дҪҚдәҺ Objects/bytesobject.c дёӯпјҡ

з”ұжӯӨеҸҜи§ҒпјҢеҪ“ Python зЁӢеәҸ ејҖе§ӢиҝҗиЎҢж—¶пјҢеӯ—з¬Ұзј“еҶІжұ жҳҜз©әзҡ„гҖӮйҡҸзқҖ еҚ•еӯ—иҠӮ bytes еҜ№иұЎзҡ„еҲӣе»әпјҢзј“еҶІжұ дёӯзҡ„еҜ№иұЎж…ўж…ўеӨҡдәҶиө·жқҘгҖӮ

еӯ—з¬ҰеҜ№иұЎ йҰ–ж¬ЎеҲӣе»әеҗҺдҫҝеңЁзј“еҶІжұ дёӯзј“еӯҳиө·жқҘпјӣеҗҺз»ӯеҶҚж¬ЎдҪҝз”Ёж—¶пјҢ Python зӣҙжҺҘд»Һзј“еҶІжұ дёӯеҸ–пјҢйҒҝе…ҚйҮҚеӨҚеҲӣе»әе’Ңй”ҖжҜҒгҖӮдёҺ е°Ҹж•ҙж•° дёҖж ·пјҢеӯ—з¬ҰеҜ№иұЎ еҸӘжңүдёәж•°дёҚеӨҡзҡ„ 256 дёӘпјҢдҪҶдҪҝз”Ёйў‘зҺҮйқһеёёй«ҳгҖӮзј“еҶІжұ жҠҖжңҜдҪңдёәдёҖз§Қ д»Ҙз©әй—ҙжҚўж—¶й—ҙ зҡ„дјҳеҢ–жүӢж®өпјҢеҸӘйңҖ иҫғе°Ҹзҡ„еҶ…еӯҳдёәд»Јд»·пјҢдҫҝеҸҜжҳҺжҳҫжҸҗеҚҮжү§иЎҢж•ҲзҺҮгҖӮ

жәҗж–Үд»¶cppеҲ°еҸҜжү§иЎҢж–Үд»¶exeзҡ„иҝҮзЁӢ#

д»ҺдёҖдёӘcppж–Үд»¶еҲ°дёҖдёӘexeж–Үд»¶пјҢеӨ§жҰӮз»ҸеҺҶдәҶд»ҘдёӢиҝҮзЁӢпјҡ

иҝҷи·ҹ еҠ жі•иҝӣдҪҚжңүе…ігҖӮеҰӮжһңе…ЁйғЁ 32 дҪҚйғҪз”ЁжқҘдҝқеӯҳз»қеҜ№еҖјпјҢйӮЈд№ҲдёәдәҶдҝқиҜҒеҠ жі•дёҚжәўеҮә(дә§з”ҹиҝӣдҪҚ)пјҢйңҖиҰҒе…ҲејәеҲ¶иҪ¬жҚўжҲҗ 64 дҪҚзұ»еһӢеҗҺеңЁиҝӣиЎҢи®Ўз®—гҖӮдҪҶ зүәзүІжңҖй«ҳ 1 дҪҚеҗҺпјҢеҠ жі•иҝҗз®—дҫҝдёҚз”ЁжӢ…еҝғиҝӣдҪҚжәўеҮәдәҶгҖӮйӮЈд№ҲпјҢдёәд»Җд№Ҳ Python зүәзүІжңҖй«ҳ 2 дҪҚе‘ўпјҹеә”иҜҘжҳҜ дёәдәҶе’Ң 16 дҪҚж•ҙж•°ж–№жЎҲз»ҹдёҖиө·жқҘпјҡеҰӮжһңйҖүз”Ё 16 дҪҚж•ҙж•°дҪңдёәж•°з»„пјҢ Python еҲҷеҸӘдҪҝз”Ёе…¶дёӯ 15 дҪҚгҖӮ

иҮідәҺдёәд»Җд№ҲйҖүжӢ©йқҷжҖҒзј“еӯҳд»Һ -5 еҲ° 256 д№Ӣй—ҙзҡ„е°Ҹж•ҙж•°пјҢдё»иҰҒжҳҜеҮәдәҺжҹҗз§Қ жқғиЎЎ пјҡиҝҷдёӘиҢғеӣҙеҶ…зҡ„ж•ҙж•°дҪҝз”Ё йў‘зҺҮеҫҲй«ҳ пјҢиҖҢзј“еӯҳиҝҷдәӣе°Ҹж•ҙж•°зҡ„ еҶ…еӯҳејҖй”ҖзӣёеҜ№еҸҜжҺ§ гҖӮеҫҲеӨҡзЁӢеәҸејҖеҸ‘еңәжҷҜйғҪжІЎжңүеӣәе®ҡзҡ„жӯЈзЎ®зӯ”жЎҲпјҢйңҖиҰҒж №жҚ®е®һйҷ…жғ…еҶөе№іиЎЎеҲ©ејҠгҖӮ

зҗҶи§ЈдәҶйқҷжҖҒеҜ№иұЎжұ пјҢеҰӮдёӢзҺ°иұЎе°ұеҫҲеҘҪзҗҶи§ЈдәҶпјҡ
